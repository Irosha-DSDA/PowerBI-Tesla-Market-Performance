# Tesla Market Performance Dashboard in Power BI

This project visualizes Tesla company market performance and sales patterns using Power BI.

## Overview

-Data source: https://lnkd.in/gmVW9gyq
-Tools: Power BI Desktop

## Features
-Interactive Power BI dashboard

-Trend analysis over time

-Volume by the period

-Year closely / Highest / Lowest price by year

-Easy to use filters and slicers
